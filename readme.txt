Обязательно:
- Сверстать страницу максимально близко к дизайну, но не увлекаться pixel perfect для всех браузеров.
- Не использовать фреймворки для вёрстки.
- При скролле вниз - шапка-навигация должна крепиться к верхнему краю окна.
- При клике на пункт меню в шапке, должен быть плавный скроллинг к соответствующей секции. В дизайне не для всех пунктов меню есть разделы.
- На первом экране кнопка «Вниз» плавно скроллит к следующему разделу.
- Любое видео встроить с ютуба так, чтобы тянулось на всю ширину экрана.
- Респонсив верстка. Самим продумать как адаптировать страницу.
- Результат работы выложить на GitHub так, чтобы можно было посмотреть код и саму верстку в браузере (github pages) и прислать ссылку.
